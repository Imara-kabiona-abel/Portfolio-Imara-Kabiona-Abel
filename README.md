## Publication rapide
1. Créez un dépôt GitHub (ex: `portfolio`).
2. Poussez les fichiers (`git init`, `git add .`, `git commit -m "Initial"`, `git remote add origin ...`, `git push`).
3. Activez GitHub Pages : Settings -> Pages -> Source = main / root.
4. Le site sera disponible sur `https://USERNAME.github.io/portfolio/`

Remplacez les liens et vérifiez les coordonnées si besoin.
